# EE6405 自然语言处理

## 内容

前半部分是传统NLP技术，后半部分是深度学习技术，由两个老师来讲。该课程为翻转课堂，老师会把所有授课内容录制视频发到ntulearn，学生需要在课前自学，课堂上会有一些练习和讨论。

## 持续性评估

每奇数周是概念多选题，每双数周是代码多选题，时间15分钟。由于24年1月开的第一批，至今过了2学期，目前不知道未来是否有变动，我是第一批学生，当时大家感觉挺难，但后来老师给提了提分。课堂quiz总共是20%+20%

期中考试35%，也是选择题，填空题，当初难度适中。

小组6人的project，做出一整套代码来。占比25%。说实话6人组太大了，我觉得2人做一个项目就行。如果运气好的话组里有1-2个佬，那就可以躺平了。

## 考试

没有期末考试的课。对于一学期5门课的选手来说，有效缓解期末的压力。
